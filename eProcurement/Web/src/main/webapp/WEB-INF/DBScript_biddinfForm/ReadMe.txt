
Functionality

1) Create Bidding form:

click Left Menu: eTendring--> Create Bidding form

Tender List page will come, Please click on "create bidding form" of selected tender.

Please select tender which have envelope.(Tender 1 having envelopes currently)

2) View Bidding form:

click Left Menu: eTendring--> Bidding form List

Bidding form page will come, By clicking on view link, you can see the structure of the form where you can enter values to correspondence table

Once you save all the values, you can view or edit details by clicking Edit/View Link.
